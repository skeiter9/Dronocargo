# Dronocargo by Omar Lozada(@skeiter9)

## install

```bash
yarn
```

### start Dev server

```bash
yarn start
```

### start storybook

```bash
yarn storybook
```

#### Notes

- https://headlessui.com/ is used in flavor to geta better UX for dynamic components
- SVG Icons by https://heroicons.com/
